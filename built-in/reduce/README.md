# `reduce`

The `reduce()` method executes a __reducer__ function (that you provide) on each element of the array, resulting in single output value.

## Syntax
```js
reduceArrayTo(
   callback( 
      accumulator, 
      currentValue
      [, index
      [, array]]
   )[, initialValue]
)
```
